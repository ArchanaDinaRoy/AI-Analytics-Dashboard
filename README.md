# AI Analytics Dashboard

A full-stack AI-powered Analytics Dashboard built using Spring Boot, MySQL, HTML, CSS, JavaScript, and Gemini AI.

## Features

- JWT Authentication
- Dashboard Analytics
- Order Management (CRUD)
- AI Assistant (Gemini)
- Charts & Reports
- Search, Sorting & Pagination
- Responsive UI

## Technologies Used

- Java 21
- Spring Boot
- Spring Security
- JWT
- MySQL
- HTML
- CSS
- JavaScript
- Gemini API
- Git & GitHub

## Setup

1. Clone the repository
2. Configure `application.properties`
3. Run MySQL
4. Start the Spring Boot application
